> Portafolio UI/UX enfocado en sistemas de diseño y arquitectura de información
> # Portafolio UI/UX — Brenda Vega

Este proyecto es mi portafolio personal enfocado en UI/UX, desarrollado como una página web estática.

## 🎯 Contexto

Mi portafolio anterior ya no representaba lo que hago actualmente.  
Decidí reconstruirlo desde cero, no como PDF ni prototipo en Figma, sino como una página real publicada.

## 🧠 Enfoque

El objetivo no fue solo mostrar diseño visual, sino reflejar:

- Cómo estructuro información  
- Cómo tomo decisiones de producto  
- Cómo pienso en sistemas, no solo pantallas  

## ⚙️ Proceso

El desarrollo fue iterativo:

- Generación base en HTML con ayuda de IA  
- Ajustes de estructura, contenido y diseño  
- Optimización de visuales para diferentes tamaños de pantalla  
- Publicación usando GitHub Pages  

## 🛠️ Herramientas

- HTML / CSS  
- Figma  
- Notion  
- Claude (generación base)  
- ChatGPT (ajustes y decisiones)

## 🚀 Deploy

El sitio está publicado usando GitHub Pages.
